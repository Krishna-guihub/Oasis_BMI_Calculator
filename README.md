# BMI Calculator

## Overview

This is a *Python-based Body Mass Index (BMI) Calculator* that calculates a user's BMI using their weight and height (in pounds and inches) and provides a health classification based on standard BMI categories.

## Features

- Calculates BMI using the formula:  
  \[
  BMI = \frac{weight \times 703}{height^2}
  \]  
  where weight is in pounds and height is in inches.
- Provides classification of BMI into:
  - Underweight
  - Normal weight
  - Overweight
  - Obese
  - Severely obese
  - Morbidly obese
- Command-Line Interface (CLI) for easy interaction.
- User-friendly prompts for input.

## Prerequisites

- Python 3.x installed on your system.
